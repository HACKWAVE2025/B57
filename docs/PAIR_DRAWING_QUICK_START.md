## ğŸš¨ Common Issues

### "I can't draw"
**Solution**: Check if you're a **Viewer**. Ask session creator to make you a **Drawer**.

### "My drawings aren't showing"
**Solution**: 
1. Check internet connection
2. Refresh the page
3. Rejoin the session

### "Canvas is too crowded"
**Solution**:
1. Save current work as snapshot
2. Click **Clear Canvas**
3. Start fresh section

### "Can't see other cursors"
**Solution**: Cursors only show when others are moving their mouse on the canvas.

## âš¡ Pro Tips

### Speed Tips
- Keep tools palette visible

- Communicate plans in chat

- Save before switching focus

- Leave whitespace for clarity
- Number steps in diagrams
